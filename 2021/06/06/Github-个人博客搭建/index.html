连接GitHub

在连接GitHub之前，需要在GitHub上创建项目

1，创建项目

首先点击右上角，头像旁边的“＋”号

然后选择弹出菜单的第一项

之后在Repository name里输入与自己账户名相同的仓库名，格式为username.github.io（必须严格按照此格式，username 没有特殊改动的话就是仓库名前面那个Owner）

因为我已经创建过了,所以显示的是项目名称已经存在了

2，下载sublime text 3

打开浏览器输入Sublime Text 3，下载

下载好之后打开创建的blog文件夹，找到_config.yml文件，用sublime text3打开

编辑文档末的字段，如果没有，就添加上去

deploy:
type: git
repo: 你的仓库地址（查找方法如下图）
branch: main （主分支名字)

因为我这里只有一只分支，默认为main就行。

修好之后按ctrl+s保存

添加SSH Key

GitHub上连接已有仓库是验证是通过SSH的公开密钥进行认证的，在这里我们来生成我们自己的SSH Key，输入下面的命令，输入完后回车，会让你输入密钥的文件名和密码，系统会在/Users/your_user_directory/.ssh/x下生成私有密钥id_rsa和公开密钥id_rsa.pub

完成以上步骤后会生成密钥，在C:\Users\Administrator\ .ssh目录下。

id_rsa文件内容是这样的，由*—–BEGIN RSA PRIVATE KEY—–开始，和—–END RSA PRIVATE KEY—–*结束

id_rsa.pub的内容是这样的，由ssh-rsa开头的

文本编辑器打开id_rsa.pub并复制里面所有内容。

然后我们在GitHub上面添加本地SSH密钥信息，在右上角头像菜单选择Settings进入设置界面

然后选中SSH and GPG keys，点击New SSH key

Title可以随便填，打开公开密钥id_rsa.pub，复制里面的内容到Key里面，填完密钥后，点Add SSH key就OK了。
再次打开Git Base，输入命令行ssh -T git@github.com

出现Hi xxx！的提示代表完成链接。
